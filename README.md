# Business X-Ray Design System

The shared design language for the Business X-Ray ecosystem.

## Purpose

This repository defines the visual and interaction standards used across:

- `bx-theme`
- `bx-platform`
- reports
- client portal
- admin dashboards
- marketing pages
- future products such as Content Intelligence

## Design Philosophy

> Clarity. Confidence. Calm.

Business X-Ray should feel premium, trustworthy and easy to understand. It should reduce anxiety, not add to it.
